Linux SCSI target framework (tgt) project

Linux target framework (tgt) aims to simplify various SCSI target driver (iSCSI, Fibre Channel, RDMA, etc) creation and maintenance. Our key goal is implementing a great portion of tgt in user space.

The tgt code (kernel-space and user-space) is available on the subversion repository. We will move the kernel-space code to scsi-target git tree shortly.

Target drivers

  • Virtual SCSI target driver in IBM pSeries

  • Qlogic qla2xxx FC target driver (not yet)

  • LSI logic fusion FC target driver (not yet)

  • iSCSI software target driver for Ethernet NICs (not yet)

  • Qlogic qla4xxx iSCSI target driver (not yet)
